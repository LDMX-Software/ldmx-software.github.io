Detailed Description

template<unsigned int m0, unsigned int m1 = 0, unsigned int m2 = 0>
class ldmx::PackedIndex< m0, m1, m2 >

A maximally-packed index of upto four different fields.

Template Parameters

in] m0 modulus of field 0 (i.e. its maximum value)

Template Parameters

in] m1 modulus of field 1 (i.e. its maximum value) optional

Template Parameters

in] m2 modulus of field 2 (i.e. its maximum value) optional
