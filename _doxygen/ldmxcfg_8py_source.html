264  def setCompression(self,algorithm,level=9):
265  """set the compression settings for any output files in this process
266 
267  We combine the compression settings here in the same way that ROOT
268  does. This allows the compression settings to be passed along as
269  one integer rather than two without any loss of generality.
270 
271  Look at ROOT's documentation for TFile to learn more
272  about the different compression algorithms and levels available
273  (as well as what integers to use). There is a summary table
274  below.
275 
276  Algorithm |int| root version
277  --------------------|---|-------------
278  ROOT global default | 0 | root-6.16
279  ZLIB | 1 | root-6.16
280  LZMA | 2 | root-6.16
281  Old (ROOT 5) | 3 | root-6.16
282  LZ4 | 4 | root-6.16
283  ZSTD | 5 | root-6.20
284 
285  Level 0 : no compression is applied
286  Level 9 : maximum amount of compression available from algorithm
287 
288  Parameters
289  ----------
290  algorithm : int
291  flag for the algorithm to use
292  level : int
293  flag for the level of compression to use
294  """
295 
296  self.compressionSetting = algorithm*100 + level
