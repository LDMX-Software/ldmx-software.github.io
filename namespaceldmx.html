@type EventBusPassenger Allows all event bus passenger objects to be handled under one name (without an inheritance tree).

Note
Any object that you want to be put into the event bus needs to be listed here as well as above.
There are two types of EventBusPassengers: 1) Those that exist inside of an STL collection (e.g. SimParticle or EcalHit). These must have Print and operator< (for sorting) methods defined. 2) Those that exist outside an STL collection (e.g. EcalVetoResult) These must have Print and Clear methods defined.
