void ldmx::Process::addDropKeepRule ( const std::string &  rule)

Add a rule for keeping/dropping event products

Note
Rules have the format: "keep/drop [name]_[pass]" Either or both of [name] and [pass] can be wildcards '*'. To keep all branches, use "keep *". Rules are processed in order and can overrule each other. Therefore, to keep only the simTracks from the "sim" pass of processing : "drop *_sim" then "keep simTracks_sim".
